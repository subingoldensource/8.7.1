<h3 daisyLanguage="1" daisyBranch="392" daisyDocument="16506-GSC" id="dsy16506-GSC" daisyNumber="1.7.2" daisyPartialNumber="2" daisyRawNumber="2">1.7.2 External Domain Value Screen</h3>


<p>Each External Domain Value record specifies a domain value that either
represents a value provided by an external source or provides a
language-specific synonym for an existing Internal Domain Value record. The
relationship to the Internal Domain Value is optional so that domain values
corresponding to external fields can be recorded without having to link the
values to an Internal Domain Value record.</p>


<p>If the current External Field Definition is not an Issue Type or an Issue
Action Type, use this page to view or define zero, one or more valid values for
the current External Field Definition, and to define the existing Domain Value
in the Internal Domain Value (IDMV) table to which it will be translated. Your
entry on this page is used during the translation process.</p>


<p>Click <strong>Add </strong>to access the fields on the External Domain Value
page.</p>

<p>Each External Issue Type record specifies either an external data source's

<p>If the current External Field Definition is an Issue Type, use this page to
